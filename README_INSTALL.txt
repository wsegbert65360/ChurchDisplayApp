Church Display App - Installation Instructions
============================================

📁 INSTALLATION:

1. Extract all files from the ZIP package to a temporary folder
2. Right-click on "INSTALL.bat" and select "Run as administrator"
   (Or use "INSTALL_USER.bat" for a non-admin installation in your Documents folder)
3. Follow the on-screen prompts
4. The app will be installed and shortcuts will be created

🖥️ WHAT GETS INSTALLED:

- Main application: ChurchDisplayApp.exe
- Desktop shortcut for easy access
- Start Menu shortcut under "Church Display App"
- Installation folder: C:\Program Files\ChurchDisplayApp\

🎯 FEATURES:

✅ Background music with pulsing visual feedback
✅ Media controls with solid color indication
✅ Working progress bar for video/audio playback
✅ Perfect image scaling (fits screen without cropping)
✅ Playlist management with drag-and-drop support
✅ Visual hint color for empty playlist guidance
✅ Toggle display feature to hide/show live window
✅ Multi-monitor support with automatic detection

🎮 USAGE:

- Add files using "📁 Add Files..." button or drag-and-drop
- Double-click items to play them on the display
- Use "🎬 Go Live" button to launch selected media
- Control playback with Previous, Play/Pause, Stop, Next buttons
- Adjust volume with the volume slider
- Use "🖼️ Blank Display" to clear the screen
- Use "🖥️ Toggle Display" to hide/show the live window
- Set up background music with the controls on the right

🗑️ UNINSTALLATION:

1. Right-click on "UNINSTALL.bat" and select "Run as administrator"
2. Follow the prompts to remove the application
3. All shortcuts and files will be removed

🛠️ TROUBLESHOOTING:

* "Missing libvlc.dll" or video doesn't play?
  You likely need the Visual C++ 2015-2022 Redistributable (x64).
  Download: https://aka.ms/vs/17/release/vc_redist.x64.exe

* "Access Denied" during installation?
  Make sure you right-click on "INSTALL.bat" and select "Run as administrator".

* App won't start after installation?
  Try running "RUN_PORTABLE.bat" to see if it works without installation.
