Church Display App - Version 1.0
=====================================

A professional church service display application for managing and presenting
media during worship services.

QUICK START
-----------
1. Run ChurchDisplayApp.exe
2. Select background music when prompted
3. Add media files to your playlist
4. Configure your display (projector/monitor)
5. Start your service!

FEATURES
--------
• Service Elements: Organized worship service components
• Playlist Management: Drag & drop files, right-click delete
• File Type Icons: Visual indicators for videos, audio, and images
• Background Music: Auto-pauses during media playback
• Multi-Monitor Support: Automatic projector detection
• Professional Interface: Clean, church-appropriate design

SYSTEM REQUIREMENTS
-------------------
• Windows 10 or Windows 11
• DirectX compatible graphics card
• Speakers for audio playback
• Projector or external display (recommended)

FILE TYPES SUPPORTED
--------------------
• Videos: .mp4, .mov, .wmv, .mkv
• Audio: .mp3, .wav, .flac, .wma
• Images: .jpg, .jpeg, .png, .bmp, .gif

TROUBLESHOOTING
---------------
• If the app won't start, ensure Windows is up to date
• For display issues, check projector connections
• Audio problems: Check speaker connections and volume settings

SUPPORT
-------
For technical support, contact your church IT department.
